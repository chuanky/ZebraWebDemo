激情之心

康斯丹海岛酒店集团集众多奢华酒店和渡假村于一体，带您领略印度洋沿岸风光，体验理想胜地的完美假日。

渡假村和酒店的四周环绕着蓝绿色的环礁湖，更有专业人员为您带来贴心服务。从奢华浮光到鲁宾逊漂流逃脱之旅，您将恣意体验永恒的冒险。

烹饪

康斯丹的美食哲学无视烹饪的地域差异，融合了自然的多样性，通过新鲜的产品、创意的技巧、和谐的风味，呈献出制作精美的食物。葡萄酒爱好者还可以在品尝美食的同时，酌饮我们独家酒窖中的特色葡萄酒。
康斯丹每一处酒店的烹饪都融合了少量地方菜肴元素，为您开启别样的味蕾体验。

不容错过的美食活动
第12届贝纳.卢瓦索厨艺大赛
2017年3月25日至2017年4月2日

查看烹饪体验 →

潜水

如果您来到印度洋度假，领略着康斯丹海岛酒店周边晶莹剔透的水域风景，却没有探索水下的海底世界，就不能说拥有完整的旅程。
无论您想要提高技巧，增加自信心，还是单纯想要娱乐，通过潜水专家的指导，在这片能见度极佳的水域中，您会惊叹于众多珊瑚礁上的生物。
从魔鬼鱼到鲸鲨，从海龟到海豚，您的印度洋海洋生物之旅将终生难忘。

查看水族乐园 →

康斯丹U Spa水疗

快乐是健康状态的情感表达......
来到这里，您将彻底放松下来，全身心感受这段幸福的旅程。
康斯丹U Spa水疗为您量身定制，旨在为您打造快乐体验。我们所有Spa水疗设备皆经过精心设计，以助您活跃身体和心灵，达到健康精神的状态，让您保持恒久的欢愉。

婚礼&蜜月

来到康斯丹海岛酒店，置身纯白的海滩之上，面对如水晶般清澈的海水或在摇摆的棕榈树下，说出那声“我愿意”。您只需在这些世界著名景点中进行选择：天堂般的毛里求斯、奢华的塞舌尔、魔幻般的马尔代夫和原生态的马达加斯加。
享受浪漫的避世之旅，抛却一切顾虑，让康斯丹酒店专业的服务团队为您安排一切。

查看婚礼&蜜月体验 →

高尔夫

豪华度假体验和锦标赛高尔夫球...
我们在3个酒店和度假村分设了3个经PGA核准的18洞锦标赛高尔夫球场：

林克斯（Links）高尔夫球场 - 康斯丹贝尔玛尔度假村和康斯丹毛里求斯王子度假酒店
传奇（Legend）高尔夫球场 - 康斯丹贝尔玛尔度假村和康斯丹毛里求斯王子度假酒店
蕾沐莉亚高尔夫球场 - 康斯丹塞舌尔蕾沐莉亚度假酒店

在毛里求斯度假 - 无限畅享高尔夫球场。
更有Marc Farry高尔夫学院专业高尔夫人员提供指导.


注明活动时间从2016年12月5日至2016年12月11日
